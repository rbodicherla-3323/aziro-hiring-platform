Use Reports with this step-by-step flow:

  1. Set period first: Pick Today/Last 24h/Last 7d/Last 28d/Custom Date to scope your generated candidates.
  2. Filter candidates: Use role and search to narrow the report list to relevant candidates.
  3. Open candidate details: Click a candidate row to expand details, and click the same row again to collapse.
  4. Validate scores: Review overall score and round-level skill breakdown before sharing outcomes.
  5. Generate artifacts: Use Generate Report when report files are not yet available.
  6. Create drive-level insights: Use Consolidated Summary to select a filtered candidate batch and produce one summary across the group.
  7. Consume outputs: Use View Report, Download PDF, and View Screenshots.
  8. Maintain freshness: Use Regenerate Report after data updates to keep files current.
